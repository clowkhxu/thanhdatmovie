# 1. Hướng dẫn setup dự án

- 1/ Clone dự án

```bash
  https://github.com/phohoccode/phoflix-v2-frontend.git
```

- 2/ Cài đặt node_modules

```bash
  npm i
```

- 3/ Chạy dự án

```bash
  npm start
```
